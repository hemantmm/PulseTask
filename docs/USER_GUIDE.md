# User Manual

## Registration and Login

1. Open frontend app.
2. Register with tenant key and role.
3. Login using email/password.

## Dashboard Usage

- Editors/Admins can upload videos.
- Viewers can monitor status and play processed videos.
- Use filters to narrow by status/sensitivity.

## Upload Flow

1. Enter title and optional metadata.
2. Select a video file.
3. Submit upload.
4. Track upload progress.
5. Wait for processing updates in real-time.

## Sensitivity Status

- `unknown`: analysis pending
- `safe`: accepted as non-sensitive
- `flagged`: potentially sensitive

## Streaming

- Click `Play` when status is `processed`.
- Built-in player streams media directly from backend.

## Admin Features

- Navigate to `Users` section.
- Create viewer/editor/admin users in current tenant.
- View all tenant users.
